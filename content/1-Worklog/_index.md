The Worklog section records the work journal throughout the **12 weeks** of the internship in the **Workforce Bootcamp - First Cloud AI Journey** program. Each week focuses on a topic on the AWS platform, following naturally from one to the next: from the initial foundational knowledge, through building network infrastructure and deploying applications, to monitoring, backup, advanced networking, and centralized operations; the final three weeks are dedicated to finalizing, testing, optimizing, and handing over a complete system.

Detailed content of each week:

**Week 1:** [Getting started with AWS: Console, IAM, Amazon S3, and EC2](1.1-week1/)

**Week 2:** [Building Amazon VPC network architecture and AWS Site-to-Site VPN](1.2-week2/)

**Week 3:** [Amazon EC2 lifecycle (Windows/Linux) and web application deployment](1.3-week3/)

**Week 4:** [Amazon RDS database and the two-tier application model](1.4-week4/)

**Week 5:** [Monitoring with Amazon CloudWatch and AWS CloudFormation](1.5-week5/)

**Week 6:** [Backup and restore with AWS Backup](1.6-week6/)

**Week 7:** [Private connectivity between VPCs with VPC Peering](1.7-week7/)

**Week 8:** [Centralized multi-VPC connectivity with AWS Transit Gateway](1.8-week8/)

**Week 9:** [Centralized server operations with AWS Systems Manager](1.9-week9/)

**Week 10:** [Finalizing and integrating the whole system](1.10-week10/)

**Week 11:** [Testing, debugging, and optimizing the system](1.11-week11/)

**Week 12:** [Documentation review, evaluation, and handover](1.12-week12/)
